📬 Contact Genesis Radio

If you’ve tried the troubleshooting guides and still can’t get things working, we’re here to help.

💬 Fastest Options

  • Mastodon: @administration@chatwithus.live
  • E-Mail: Genesis Hosting Technologies

📄 What to Include

If you email or message us, try to include the following so we can help faster:

  • Your device (phone, PC, tablet)
  • Where you’re tuning in from (roughly — city/country)
  • Are you using Wi-Fi, mobile data, or VPN?
  • What you’ve tried already
  • Any error messages (if applicable)

🕓 When to Expect a Response

We usually reply within 24 hours, often faster. If it’s a known issue, we’ll point you to the fix. If it’s something new, we’ll investigate.
