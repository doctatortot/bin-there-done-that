🔗 Direct Stream Links

If you’re having trouble with the web player, you can use these direct stream URLs in any compatible audio app or browser. No frills — just the music.

🎧 Main Streams

Use these in VLC, Winamp, myTuner, XiiaLive, or your browser’s address bar:

http://stream.genesis-radio.net:7454/stream

🧪 Compatibility

  • Browsers like Firefox and Chrome can play this directly.
  • iOS and Android may prompt for a media app — that’s expected.

📥 How to Use

  • VLC: Media → Open Network Stream → Paste the URL
  • Winamp/Foobar: File → Open URL
  • Mobile: Use the URL in VLC or XiiaLive

🌐 Alternate Access

If these links fail, you might be affected by network restrictions or VPN routing.
